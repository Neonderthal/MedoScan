# 🏥 MedoScan — Patient Driven Insights

> AI-powered social listening for real-world medicine insights.

🌐 **Live Demo:** [https://medoscan.vercel.app/](https://medoscan.vercel.app/)

---

## 📌 About

MedoScan is an AI-powered platform that analyzes real-world patient experiences and social data to generate meaningful medical insights. It helps researchers, healthcare professionals, and patients understand how medicines are perceived and experienced in the real world.

---

## ✨ Features

- 🤖 AI-powered analysis of patient-reported data
- 📊 Real-world medicine insights dashboard
- 🔍 Social listening for drug and treatment feedback
- 📈 Data visualization and trend tracking
- ⚡ Fast and responsive React-based UI

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Python (FastAPI / Flask)
- **AI/ML:** Custom NLP pipeline
- **Deployment:** Vercel

---

## 🚀 Getting Started

### Prerequisites
- Node.js >= 14
- Python >= 3.8

### Installation

```bash
# Clone the repository
git clone https://github.com/Neonderthal/MedoScan.git

# Navigate to the project folder
cd MedoScan

# Install frontend dependencies
npm install

# Install backend dependencies
pip install -r requirements.txt
```

### Running the App

```bash
# Start the frontend
npm start

# Start the backend (in a separate terminal)
python server.py
```
